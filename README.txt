Что делает этот бот? 

- Отправляет приветственное сообщение при старте бота.
- Отправляет случайные смешные GIF по команде.
- Эхо-ответ на любые текстовые сообщения.

Команды

- `/start` - Получить приветственное сообщение.
- `/funnygif` - Получить случайный смешной GIF.

Установка и запуск

Для работы с ботом вам потребуется:

- Docker
- Telegram bot token

Конфигурация

1. Создайте файл `.env` в корне проекта и добавьте в него ваш Telegram bot token:

    ```
    TOKEN=ваш_токен_бота_здесь
    ```

2. При необходимости настройте `GIPHY_API_KEY` в коде бота.

Сборка Docker образа

Перейдите в директорию проекта и выполните следующую команду для сборки Docker образа:

```sh
docker build -t telegram-bot .
```

Запуск бота

Запустите бота с помощью Docker, используя следующую команду:

```sh
docker run -d --env-file .env telegram-bot
```
